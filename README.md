# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行车道线实例分割检测
- 提供完整源码、数据集、训练好的模型及训练结果
- 内置UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存功能
- 可自定义置信度、IOU阈值
- 提供多种检测模型选择
- 支持选择显示标签和原图

## 适用场景
- 道路交通监控与智能分析
- 自动驾驶辅助系统开发
- 车道线检测算法研究
- 交通场景图像识别
- 实时视频监控处理
- 批量图片分析处理

## 技术特点
- 基于YOLOv8实例分割算法
- 使用PyCharm+Anaconda或VSCode+Anaconda开发环境
- 依赖Python3.9、OpenCV、PyQt5、Ultralytics、Torch1.9等技术栈
- 包含8000张车道线标注数据集
- 提供完整的项目运行教程和界面源文件
- 支持多种输入源的实时检测功能
